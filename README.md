# HR Employee Attrition Analysis

## Project Overview
Analysis of IBM HR dataset to identify 
key factors driving employee attrition.

## Dataset
- Source: IBM HR Analytics Dataset
- Size: 1470 employees, 35 features

## Key Findings
1. Overall attrition rate is 16.1%

2. Sales department has highest attrition
   at 20.6% vs company average 16.1%

3. Employees working overtime are 3x more
   likely to leave (30.5% vs 10.4%)

4. Young employees aged 18-30 show
   highest attrition rate

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Business Recommendations
1. Reduce overtime in Sales department
2. Create retention programs for
   employees under 30
3. Investigate Sales compensation package
